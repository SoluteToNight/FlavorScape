### **一、 核心视觉基调与技术/数据选型**

#### **1\. 视觉基调 (极致纯净博物志风)**

* **全局色彩：** 坚持明亮、通透的浅色调（米白、浅灰为主）。彻底剔除冗杂的 UI 控件，让自然物理地球本身成为绝对的视觉主干。  
* **UI 材质：** 左右两侧的数据面板采用高透明度毛玻璃（80% 磨砂透白），边缘采用无感渐变融入背景，实现“悬浮且不可见边框”的沉浸感。  
* **数据高亮色：** 动效光斑采用极高明度、高纯度的自然提取色（如胭脂红、松石绿、琥珀黄），在素色的自然底图上形成清晰干脆的视觉刺透力。

#### **2\. 技术栈架构**

* **前端框架：** React 或 Vue 3。  
* **WebGIS 渲染引擎：** **Deck.gl**（核心调用 PathLayer 渲染暗线拓扑，调用 TripsLayer 渲染具有物理轨迹的流光粒子）。  
* **图表渲染：** ECharts 或 D3.js。

#### **3\. 权威开放数据源 (物理与生态底座)**

* **唯一物理底图 (L0)：** **Natural Earth Data**。提供最纯粹的地形起伏、海岸线和主干水系，绝对剔除现代国界线。结合 MapLibre GL JS 进行渲染。  
* **历史锚点网 (L0 叠加)：** **CHGIS (中国历史地理信息系统)**。提取特定纪元（如明清）的古水系（大运河、古黄河）与核心府州坐标，以极淡的灰线/灰点隐入底图。  
* **全球生态区划 (L1 核心)：** **WWF TEOW (世界自然基金会全球陆地生态区划)**。无视人类政治边界，提供全球 800+ 个绝对客观的自然生态区边界（例如精确勾勒南美安第斯山脉的热带雨林区）。  
* **国内高精区划 (L1 补充)：** **中科院 RESDC (中国生态地理区域数据)**。用于对国内区域（如四川盆地、塔里木盆地）进行极其精准的亚区物理边界刻画。

**二、 空间可视化图层方案 (Deck.gl 渲染逻辑)**

在这个明亮的 2.5D 视锥体中，抛弃所有的生硬色块，完全用“线、光、隐形边界”来解构时空。

* **L0 纯净物理底图层：** Natural Earth 浅色地形阴影 \+ CHGIS 灰色历史脉络。  
* **L1 自然生态单元层 (矢量线框)：** 读取 WWF TEOW 或 RESDC 的精确多边形（Polygon）。**多边形内部完全透明（Fill Opacity \= 0）**，仅用极细的浅褐色实线（1px）勾勒出地貌/生态区的物理轮廓。所有复杂的温湿度数据隐藏于 GeoJSON 属性中，静待检索。  
* **L2 脉冲流光传播层 (光效叙事核心)：**  
  * **基础轨道 (静态暗线)：** 跨越洲际的物种传播路径平时以低透明度、低饱和度的暗灰色细线蛰伏于地表（虚线表示越洋，实线表示陆路/内河）。  
  * **传播方向 (流光游走)：** 高亮的光效粒子（Light Trail）沿着暗线滑动。光斑向哪个坐标移动，即代表基因向哪方迁徙。  
  * **先后顺序 (时序错落接力)：** 彻底废除时间轴面板。利用代码控制光斑的**触发时机**：年代早的事件（如大航海时代跨越大西洋）率先点亮并滑动；年代晚的事件（如顺长江进入四川）在随后的数秒内接力亮起。通过视觉上的先后发车，直观表达历史编年史。  
* **L3 实体锚点层 (微光节点)：** 代表成品美食或原生食材的节点。悬浮于坐标正上方，呈现微弱的呼吸光晕，并在地表投射阴影。

**三、 无界 HUD 交互架构 (智能上下文面板)**

彻底抛弃繁琐的过滤器和底部时间切片器。UI 仅保留左右两侧，通过用户的检索行为，**由系统智能计算并呈现相关图层**。

#### **1\. 左侧：极简探索台 (约占宽 20%)**

* **全局入口 (Omni-Search)：** 屏幕左上角仅漂浮一个纯白无框的搜索输入框，带微弱阴影。  
* **智能图层触发 (Smart Context Trigger)：** 默认状态下，地图全空，只有大好河山。当用户搜索某项内容时，引擎在后台通过图数据库查询出最短逻辑路径。随即，地图上**仅显现**与搜索词强关联的 L1 生态边界和 L2 暗线轨道及光效。

#### **2\. 底部：全空视角**

* **留白美学：** 砍掉所有控件。下方完全透空，把最大面积的视野留给地球轮廓和跨洋流光动效的演绎。

#### **3\. 右侧：动态对象检视器 (约占宽 25%)**

这块隐形毛玻璃面板是深度数据的承载器。它会根据高亮对象类型的不同，平滑切换展示状态：

* **当检视【L1 生态多边形】：** 面板显示“自然生态档案”。包含长期的温湿度双轴曲线图（ECharts 渲染），以及“物理环境如何筛选特定饮食基因”的推演文案。  
* **当检视【L3 成品节点】：** 面板显示核心“风味雷达图”（粗犷、辛辣、高碳水等向量），下方列举出拼凑出该表型的底层食材积木。  
* **当检视【L2 流光连线/轨迹】：** 面板展示具体的古籍文献摘录（被结构化处理过的文字映射），作为这条跨洋光线的历史文献考证。
