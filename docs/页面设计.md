# 《寻味地理》整体网站设计思路

## 一、 品牌理念与视觉哲学

**核心主张**  
“每一次风味演变，都是自然与环境写在大地上的基因突变。”  
「寻味地理」不只是一张标注美食位置的地图，而是一部**用自然生态语言解码风味演化的数字博物志**。它以地球物理为纸、数据光流为墨，让用户在纯净的视觉沉浸中，看见气候、山川、水系是如何悄然修改食物“基因”的。

**视觉基调延续——极致纯净博物志风**  
- **底色体系**：全局保持明亮的米白、浅灰与微暖的地形阴影，还原地球纯粹的呼吸感，杜绝任何装饰性噪点。  
- **界面存在感**：UI 只为“揭示”而存在，不抢夺自然本体。所有面板采用高透毛玻璃，边缘无感消隐，形成“悬浮且不可见”的视觉状态。  
- **数据高亮逻辑**：以**胭脂红、松石绿、琥珀黄**等自然提取色作为信息刺穿点，在素色的地学底图上制造清晰、果断的认知焦点。  
- **字体与排版**：使用人文主义无衬线体（如思源黑体/Inter），字重轻盈，字距疏朗，营造科学图鉴的安静呼吸感。  

**品牌基因隐喻**  
整个网站围绕“同源异构”构建叙事宇宙：  
- **底图 = 自然环境 DNA**（L0–L1 物理生态）  
- **连线 = 传播路径 RNA**（L2 光流叙事）  
- **节点 = 风味基因表达**（L3 实体锚点 + 雷达图）  
- **用户行为 = 进化选择压力**（每一次搜索都像一场自然筛选）

## 二、 网站信息架构

网站面向广大用户，需要兼顾“瞬间捕获好奇心”与“深度探索沉浸”。整体采用**一主多辅的星环式结构**：以地图探索页为绝对核心，若干轻量内容页围绕展开，并保证在任何页面都能一键回到“地图叙事”。

```
导航结构（始终悬浮，透明无边框）：
├─ 首页（瞬时引力）  
├─ 探索地图（核心体验）  ← 默认入口
├─ 风味基因库（结构化纵览）  
├─ 时空叙事馆（专题故事）  
└─ 关于·方法论（透明信用层）  
```

**设计原则**  
- **零阻断浏览**：导航栏收拢为左下角一枚极简的呼吸光点，点击后优雅展开，平时完全消失。  
- **全局 Omni-Search**：搜索是贯通所有页面的最高频动作，搜索框在首页与地图页保持同一质感与位置，支持模糊匹配食材、地域、风味维度。  
- **渐进式信息披露**：首页仅给最诱人的一句话与视觉钩子；地图页承载所有复杂数据；基因库与叙事馆提供非地图场景的知识拆解。

## 三、 页面设计详解

### 3.1 首页 —— 初始印象与瞬间直觉

**设计目标**：让用户在 3 秒内理解“这个网站在做什么”，并激发进入地图的冲动。  

**视觉构成**  
- **全屏动态背景**：一段极简的粒子动画——若干风味光点在素色大地上沿古水系暗线缓慢游走，无任何 UI 遮挡，只有左下角那枚呼吸光点暗示交互。  
- **中心一句话文案**：以极细字重和宽松字间距浮现——“**探寻同一种食物，在不同水土中写下的味道基因。**”  
- **入口按钮**：文案下方是一个悬浮的毛玻璃胶囊按钮，文字只有“**开启地图**”，带琥珀色微光。点击后页面平滑推入地图页（过渡动效采用地图从远到近的视锥拉伸，衔接 2.5D 视锥体的设定）。  
- **下方静默轮播**：极度克制的文字轮播，展示系统内的风味基因短句，如“成都·辣度 0.95 → 顺德·辣度 0.0”，暗示背后的数据宇宙。

**交互细节**  
- 页面加载瞬间，光点首先停止，再恢复运动——微小的时间停顿制造呼吸感。  
- 鼠标移动时，光点会产生极轻微的引力偏移，暗示用户“你正与数据同在一个物理空间”。

### 3.2 探索地图页 —— 核心沉浸相（已详述，整合为网站整体）

直接沿用上一轮所设计的“无界HUD + 纯净物理底图 + 三层时空图层”。在该网站框架中，地图页承担**探索与即兴推理**的核心任务，并作为其他页面跳转的枢纽。此处补充两点衔接性设计：

- **从基因库/叙事馆跳回地图时**：自动带入对应节点或故事线的观察参数，使地图直接呈出相应的 L1 生态区高亮、L2 光流激活状态，保持上下文连贯。  
- **地图页内的微型“返回首页”设计**：左下角呼吸光点保持存在，点击可随时回到首页，确保不会产生迷宫感。

### 3.3 风味基因库 —— 可纵览的基因多维矩阵

**设计目标**：为喜欢结构化浏览的用户提供一个非地图的对比视图，能够快速并排查看不同地域同一食物类别的“风味基因型”。

**页面布局**  
- **顶部**：极简筛选栏，仅有食物类别（火锅/烧烤/腌菜…）一个下拉项，其余交给智能排序。  
- **主体**：采用**卡片流 + 风味雷达图碎片**的网格布局，每张卡片代表一个地域节点。  
- **卡片设计**：  
  - 延续毛玻璃微透材质，卡片边缘无边框，仅有微弱阴影。  
  - 正面展示该节点最突出的两个风味维度的大号数值（如“麻 0.90*辣 0.95”），以及一个微型雷达图矢量图形。  
  - 悬停时卡片略微抬起，并且雷达图发生顺滑旋转，透出更多维度。  
  - 卡片背景会淡淡透出该地区对应的 L1 生态区轮廓线，将物理环境暗示直接织入卡片中。  
- **底部提示**：一行灰色小字“点击任一节点，跳转至地图观察其基因来源与传播路径”。

**交互与动效**  
- 进入页面时，卡片按经纬度自东向西依次淡入，形成地理推演的时间感。  
- 支持多选对比模式：选中两张卡片后，页面自动生成基因差异分析短句，如“陈皮替代了牛油，湿度筛选了辣度归零”。

### 3.4 时空叙事馆 —— 专题故事线

**设计目标**：以线性叙事的方式，呈现某一条最具代表性的“基因传播路线”，让用户像阅读一部多章节的视觉故事一样理解风味演化。

**页面结构**  
- **左侧章节导航**：极细的竖线串联数个光点，每个光点代表故事的一个地理/历史节点（如：大运河漕运节点→四川盆地深井盐开采→辣椒入华传播链）。  
- **右侧内容区**：采用全屏沉浸式视图，不是传统长图文，而是**一个缩小版的无边界地图空间**，自动播放该章节对应的 L2 光流粒子运动，地图下方叠加古籍引用与简短推理文案。  
- **底部控制**：仅有一个极细的进度点，无时间轴。

**体验细节**  
- 用户可上下滑动切换章节，光流粒子会随章节切换而产生空间跳跃，但通过平滑的相机过渡保证叙事不中断。  
- 最后一章自动浮现一个“打开全地图自由探索”按钮，完成从被动叙事到主动探索的自然引导。

### 3.5 关于·方法论 —— 透明信任层

**设计目标**：向用户说明数据来源与量化逻辑，建立学术与审美之间的信任。风格保持极简，但需提供清晰的可点击条目。

**页面布局**  
- 整页仍是大量的留白，中央一个毛玻璃方块承载少量必要文字：WWF生态区来源、RESDC数据、CHGIS历史数据、风味向量量化方法。  
- 每项数据源以一条暗线连接至一个迷你光点，点击光点弹出更详细的数据字段说明。  
- 底部提供“LLM辅助清洗说明”的入口，但默认折叠，感兴趣的用户可展开查看自动化配方解析的流程简述，配极简流程图，保持同样的毛玻璃视觉。

## 四、 跨页面组件与动效设计规范

### 4.1 全局搜索组件（Omni-Search）
- **位置与形态**：地图页固定左上角；首页中心偏上；其他页面顶部居中。统一为一段极细的灰色圆角输入框，背景半透，带 0.5px 阴影，无图标，仅有占位文字“输入食物、地域或风味…”。  
- **触发机制**：聚焦时背景变纯白（微弱过渡），但依然保持极简；输入后下拉面板以毛玻璃浮层展示智能关联结果，并优先匹配地图所需轨迹，选中即跳到地图页相应视角。  
- **跨页面**：在任何页面进行搜索，结果若涉及地图节点，都带一个“在地图中查看”的统一操作项。

### 4.2 导航光点组件
- **静态状态**：一个直径 8px 的圆点，使用琥珀黄，带外发光和呼吸脉动，位于左下固定位置。  
- **悬停状态**：光点周围缓慢旋转浮现出四个等距光点，分别代表首页、基因库、叙事馆、关于，这四枚光点以极细线连接，菜单文字延迟出现。  
- **移动端**：光点改为更易触碰的尺寸，但仍保持单点简约。

### 4.3 风味雷达图组件
- 在所有出现雷达图的场景（地图右侧面板、基因库卡片），统一使用 ECharts 雷达图，配置透明底色、极细浅灰网格线、数据区域填充为对应风味维度的高亮自然色渐变（如麻感配琥珀黄渐变），无图例无标题，仅靠形状表达。  
- 切换节点时，雷达图数据变化通过`animationDuration: 800`的时间变形，模拟基因突变的丝滑感。

### 4.4 光流粒子线组件
- 在首页背景、叙事页、以及偶尔出现的装饰位置，光流粒子遵循同样的物理参数：非匀速、在生态边界处微量停顿、尾迹有淡出渐隐。动效统一使用 `requestAnimationFrame` 控制流体缓动，确保全站一致。
