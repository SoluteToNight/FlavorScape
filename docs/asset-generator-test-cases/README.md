# 空间品牌资产生成器测试文档集

本文件夹用于测试 `/import` 空间品牌资产生成器的 DeepSeek 文档分析能力。

这些样本是虚构产品文档，不复制真实品牌文案；它们参考了主流年轻化消费品的品类表达，包括新式茶饮、原叶鲜奶茶、现制咖啡、生椰/植物基、无糖气泡水、冻干咖啡、酸奶碗、地域风味零食和功能化轻食。

## 使用方式

1. 打开 `/import`。
2. 选择任意案例模板。
3. 打开本目录下某个 `.md` 文件。
4. 复制 `测试文档正文` 部分到“DeepSeek 文档分析”文本框。
5. 点击“DeepSeek 分析文档”。

## 分档说明

- `low-score/`：资料模糊、证据缺失、夸张表达多，预期低分。
- `mid-score/`：有基本产品、产地、工艺信息，但指标和证据链不完整，预期中等分。
- `high-score/`：有产品结构、产地节点、工艺流程、检测指标、批次追溯和可视化素材，预期高分。

## 建议测试目标

- 资料完整度是否能区分“宣传文案”和“可验证资料”。
- 证据强度是否能识别检测指标、认证、批次、产地说明。
- 可视化适配度是否能提取地图节点、路线、时间轴、雷达图指标。
- 风险评分是否能发现夸大功效、产地含混、0糖/健康类表达不严谨等问题。
