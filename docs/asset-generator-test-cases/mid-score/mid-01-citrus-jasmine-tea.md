## 测试文档正文

产品名称：柑香茉莉冰茶  
品类：瓶装茶饮  
目标人群：喜欢清爽茶感、低甜和果香的 18-30 岁用户。

产品定位：以茉莉绿茶为茶基底，加入柑橘风味，强调清爽、低甜、花香和果香的平衡。适合便利店、校园、办公楼和夏季户外场景。

原料说明：

1. 茶基底：绿茶茶叶，供应商标注为广西、福建两地采购。
2. 花香来源：茉莉花风味浓缩液，供应商提供基础食品生产许可。
3. 果香来源：柑橘风味配料，未提供具体果汁含量。
4. 甜味系统：赤藓糖醇、少量白砂糖，目标口感低甜。

工艺流程：

1. 茶叶热水浸提。
2. 过滤后调配茉莉风味和柑橘风味。
3. UHT 杀菌。
4. 无菌冷灌装。
5. 成品抽检。

已提供资料：

- 产品配料表草案。
- 三个试制批次的感官评分表。
- 一份供应商资质扫描件。
- 计划包装设计稿。

缺失资料：

- 未提供完整营养成分表。
- 未提供茶多酚、咖啡因、总糖、微生物检测指标。
- 产地只到省级，无法精确映射基地。
- 茉莉花和柑橘风味是否来自真实原料仍需证明。

建议可视化方向：可做“茶叶采购省份 + 调配工厂 + 核心消费场景”的基础地图，但证据链不适合做高可信产地溯源。
