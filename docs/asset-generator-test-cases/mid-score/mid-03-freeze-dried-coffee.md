## 测试文档正文

产品名称：小罐冻干冷萃咖啡  
品类：冻干速溶咖啡  
目标人群：办公室、露营、旅行、学生宿舍用户。

产品定位：一颗一杯的便携冻干咖啡，适合冷水、牛奶、椰乳快速冲泡。强调便携、风味稳定、低糖和 DIY 搭配。

原料与供应：

- 咖啡豆：云南普洱和巴西阿拉比卡拼配。
- 烘焙：中度烘焙，焦糖、坚果、轻微果酸。
- 加工：上海周边代工厂进行萃取、浓缩、冻干和分装。

工艺摘要：

1. 咖啡豆拼配与烘焙。
2. 低温萃取。
3. 真空浓缩。
4. 冻干成型。
5. 小罐充氮包装。

已有资料：

- 供应商提供咖啡豆采购批次。
- 代工厂 SC 资质。
- 水分含量、菌落总数、大肠菌群抽检报告。
- 包装规格：2g/颗，12 颗/盒。

缺失资料：

- 未提供咖啡因含量。
- 未提供杯测记录原件。
- 云南和巴西拼配比例属于商业机密，无法公开。
- 产区只到普洱市，缺少庄园或合作社坐标。

可视化建议：适合做“原料双产区 + 加工工厂 + 消费场景”的供应链地图；不适合强调单一庄园风土。
