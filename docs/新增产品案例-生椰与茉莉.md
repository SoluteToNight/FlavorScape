## 案例二：广西横县 · 七窨茉莉翠芽茶底

我已提供一个效果很好的范例图，可以直接作为网站展示的素材：
"G:\code_python\Flavor_Space\FlavorScape\docs\Reference Pictures\Market\茉莉.png"

- **品牌心智坐标**：联想 霸王茶姬 / 喜茶（爆款“万里木兰”或“伯牙绝弦”的底蕴灵魂）
- **消费市场洞察**：新国风茶饮的尽头是“茶汤纯正度”。年轻人开始为“几窨工艺”和“纯茶无香精”买单。茉莉花茶自带极强的空间对流感（福建的茶坯，广西的花）。

### 1. 空间溯源节点 Map Nodes

- **【茶坯产地】**：福建宁德绿茶基地（高山清明前单芽，负责提供清爽的茶骨）。
- **【花源产地】**：广西横县现代化茉莉基地（中国茉莉之乡，负责提供高浓度精油花香）。
- **【工艺节点】**：横县数字化非遗窨制工坊（控温堆窨、机械起花）。
- **【供应链中枢】**：东莞/武汉 现代化茶饮配方供应链仓（充氮防潮密封包装）。
- **【消费终端】**：新式茶饮门店（智能化出茶机，30秒高压萃取纯茶汤）。

### 2. 工艺流程叙事 Craftsmanship Workflow

> **风味的双城记：两千公里，七次生死交融的“窨制”**
>
> 1. **两千公里的双向奔赴**：清明前采摘的福建高山绿茶，运抵广西横县，静待夏日盛放的茉莉。
> 2. **午后伏天采花**：只采摘晴天午后 2 点到 4 点、花蕾饱满如白珍珠的“当天欲开伏花”。此时精油浓度达到巅峰。
> 3. **夜间堆窨与伺花**：静待至深夜茉莉微开呈“虎爪状”。将茶坯与鲜花层层叠放。茶骨如海绵般疯狂吸收花朵吐出的香气精油，其间需多次人工“通花”散热，防止茶坯被闷熟。
> 4. **七窨一生**：每窨制一次，需“起花”（筛除旧花）并更换一批高品质鲜花。**“七窨”意味着这个过程要生死循环重复 7 次**，历时近 20 天，消耗数倍于茶重量的鲜花，方得“只闻花香，不见花影”的顶级茶底。

### 3. 工作台资产包切片 JSON Mapping

- **`evidence` (实证数据)**：气相色谱-质谱联用（GC-MS）测定的茉莉花香精油残留（芳樟醇、乙酸苄酯含量曲线图）、窨制温度控制曲线。
- **`marketingCopy` (详情页文案)**：“看不见一朵花，却咽不下整座春天的香。历经七次换花、168小时昼夜不眠的窨制，让福建的茶骨，彻底融进了广西横县的夏夜花香。”



## 全新案例：文昌生椰（冰榨原味生椰乳）

- **品牌心智坐标**：联想 瑞幸（生椰拿铁灵魂原料）、库迪、饮品店里的各种“清补凉/生椰”系列。
- **消费市场洞察**：从“生椰拿铁”常驻热销榜开始，“生椰”已经成为年轻消费者心中“清爽、植物基、低负担”的代名词。各大品牌内卷的焦点已经从“风味糖浆”变成了“100%冷榨原汁与HPP超高压灭菌技术”。

### 1. 空间溯源节点 Map Nodes

- **【原料产地】**：海南文昌 · 东郊椰林（北纬 19° 核心产区，常年强日照，含盐沙壤土，椰肉油脂香极浓）。
- **【冷榨加工】**：文昌现代产业园 · 零度生榨工厂（从破壳到出汁控制在 **1 小时**内）。
- **【锁鲜节点】**：超高压锁鲜中心（高压 HPP 灭菌，拒绝高温煮熟，保留“生”的原始风味）。
- **【跨海冷链】**：全程 **-18°C** 跨海冷链大仓（海运+陆运，直发全国华东/华南中心仓）。
- **【标杆餐桌】**：全国 10000+ 家连锁咖啡/茶饮门店（与浓缩咖啡或茶汤瞬间碰撞复配）。

### 2. 工艺流程叙事 Craftsmanship Workflow

> **1小时的“生死时速”与 600MPa 的冷压奇迹**
>
> 1. **黄金树龄老椰现采**：严格筛选文昌当地树龄 15 年以上、生长期 10-12 个月的饱满老椰。只有这个阶段的椰肉，油脂含量和月桂酸浓度才达到黄金比例。
> 2. **1小时生榨闭环**：椰子从采摘破壳、分离椰子水、刮取椰肉，到投入超低温冷榨机进行高倍复配生榨，全链路在 **1 小时**内闭环完成。多一分钟，椰脂就会面临氧化酸败的风险。
> 3. **600MPa HPP 瞬间超高压锁鲜**：颠覆传统新消费行业的超瞬时高温灭菌（UHT），采用 **HPP（High Pressure Processing）冷杀菌技术**。在超高压状态下，利用相当于 6 个马里亚纳海沟深处的压力，在不对椰乳进行任何加热的前提下瞬间杀灭细菌。这使得椰乳内部的植物蛋白和新鲜椰香被近乎完美地“定格”在刚破壳的状态。

## 一、 核心攻坚：如何在网页上实现“手绘风漫画叙事”？

网页和纸质海报最大的不同在于**交互性**和**响应式（适配不同屏幕）**。如果直接把整张大图放上去，字会看不清，且无法动态修改。我们需要把它拆解为“数据 + 组件 + 动效”。  

### 1. 第一步：切图与资产准备

让你的搭档或你自己，用图片处理工具把右侧的 **7 个步骤的手绘插画** 单独裁剪出来，保存为带有轻微投影或干净背景的 `webp` 格式（如 `step1.webp`, `step2.webp`...），放进 `src/assets/images/cases/jasmine/` 目录下。  

### 2. 第二步：将叙事内容写入 JSON 数据源

根据你们规划的 `product_cases/*.json` 结构，在茉莉花茶的 JSON 文件中拓展一个 `craftsmanship` 数组：  

JSON

```
"craftsmanship": [
  {
    "step": 1,
    "title": "茶坯溯源 · 清明采摘",
    "desc": "福建高山茶园，清明前采摘单芽，鲜叶鲜嫩，茶骨清爽。",
    "image": "@/assets/images/cases/jasmine/step1.webp"
  },
  {
    "step": 2,
    "title": "伏天采花 · 午后精华",
    "desc": "晴天午后 2-4 点，只采‘伏花’，花蕾饱满，精油浓度最高。"
  }
  // ... 依次类推写满 7 个步骤
]
```

### 3. 第三步：前端 Vue 3 交互式组件实现

在网页端，海报里这种“之字形（Zigzag）”的连线在手机端或不同分辨率下很容易错位。**推荐使用“滚动聚焦漫画墙（Scroll-Active Comic Grid）”或“横向叙事传送带（Horizontal Storytelling）”来实现。**

这里为你提供一个标准的 Vue 3 布局思路，利用 CSS Grid 完美复刻海报的 4+3 漫画网格，并加上动态连线特效：

### 1. 左侧墨绿区域 ➡️ 映射到 `/marketing`（营销海报模块）

- **现状与改造**：你们在规划中提到要新增“新消费品牌风”和“茶饮供应链风”模板。**请直接把左侧的排版做成一个 Vue 模板组件**！  
- **实现方式**：
  - 左上角放置动态商品主图。
  - 左下角用 CSS Flex 布局做成参数网格（产地、工艺、茶坯等），下方整齐排列你们设计的 5 个小图标（非遗工艺、产地直供等）。
